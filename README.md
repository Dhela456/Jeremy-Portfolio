## About Me

I am a data scientist with a passion for uncovering the stories hidden inside complex datasets. My work spans the full data science pipeline — from exploratory analysis and feature engineering to machine learning model development, evaluation, and deployment.

I specialize in:

- **Predictive Modeling** — building and tuning supervised ML models for regression and classification problems
- **Time Series Forecasting** — applying ARIMA, Prophet, and deep learning approaches to sequential data
- **Unsupervised Learning** — customer segmentation and pattern discovery via clustering techniques
- **Data Visualization & Dashboards** — turning analytical outputs into clear, decision-ready visuals

Currently deepening expertise in Deep Learning, MLOps, and cloud-based model deployment.
